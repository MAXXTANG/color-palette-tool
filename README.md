# 🎨 圖片色票提取器 | Color Palette Generator

一款專為社群小編、設計師與圖文創作者打造的輕量級網頁工具。只需上傳圖片，系統將自動為您解析出最具代表性的 5 款色彩，幫助您快速決定排版配色！

## ✨ 核心特色

*   **🧠 智能色彩提取**：使用 `Color Thief` 演算法，快速且精準地抓取圖片主要色調。
    
*   **📋 一鍵快速複製**：點擊色塊即可自動複製 HEX 色碼，附帶滑順的 Toast 提示動畫。
    
*   **🖱️ 直覺上傳操作**：支援 macOS / Windows 原生的「拖曳上傳」及傳統點擊上傳。
    
*   **🌌 現代深色質感**：採用深邃石板黑搭配「毛玻璃」特效，讓色彩展示更具高級感。
    
*   **📱 完美適應螢幕**：基於 Tailwind CSS 打造的響應式設計 (RWD)，手機、平板、電腦皆可順暢使用。
    

## 🛠️ 技術棧

本專案採用最輕量、免編譯的 Vanilla 前端技術打造（單一 HTML 檔案）：

*   **核心架構**：`HTML5` + `Vanilla JavaScript`
    
*   **UI 排版**：`Tailwind CSS` (via CDN)
    
*   **字體視覺**：`Google Fonts (Inter)`
    
*   **色彩解析**：[`Color Thief`](https://www.google.com/search?q=https://lokeshdhakar.com/projects/color-thief/ "null")
    

## 🚀 如何在本地端執行

這是一個無需任何伺服器或建置工具的純網頁專案。

1.  將本專案 Clone 到本地端，或是直接下載 `index.html` 檔案。
    
2.  對著 `index.html` 點擊兩下，使用任何現代瀏覽器 (Chrome, Safari, Edge, Firefox) 開啟。
    
3.  拖曳一張圖片進去，立刻開始使用！

A lightweight web tool designed for social media managers, designers, and content creators. Simply upload an image, and the system will automatically extract the 5 most representative colors to help you quickly decide on your layout color scheme!

## ✨ Features

*   **🧠 Smart Color Extraction**: Uses the `Color Thief` algorithm to quickly and accurately extract the main tones of the image.
    
*   **📋 One-Click Copy**: Click a color swatch to automatically copy its HEX code, accompanied by a smooth Toast notification animation.
    
*   **🖱️ Intuitive Upload**: Supports native macOS / Windows "Drag & Drop" as well as traditional click-to-upload.
    
*   **🌌 Modern Dark Premium UI**: Features a deep slate black background combined with Glassmorphism effects, making the color display look premium.
    
*   **📱 Fully Responsive**: Built with Tailwind CSS for Responsive Web Design (RWD), ensuring smooth usage on mobile phones, tablets, and desktops.
    

## 🛠️ Tech Stack

This project is built using the most lightweight, build-free Vanilla front-end technology (a single HTML file):

*   **Core**: `HTML5` + `Vanilla JavaScript`
    
*   **Styling**: `Tailwind CSS` (via CDN)
    
*   **Typography**: `Google Fonts (Inter)`
    
*   **Color Extraction**: [`Color Thief`](https://www.google.com/search?q=https://lokeshdhakar.com/projects/color-thief/ "null")
    

## 🚀 How to run locally

This is a pure web project that requires no server or build tools (No-build).

1.  Clone this repository to your local machine or directly download the `index.html` file.
    
2.  Double-click `index.html` to open it in any modern browser (Chrome, Safari, Edge, Firefox).
    
3.  Drag and drop an image into it and start using it immediately!
